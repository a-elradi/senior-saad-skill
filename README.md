## What is SAAD?

**SAAD** is an elite Claude Code skill that acts as your Senior System Architect, UX Designer, Security Analyst, Product Strategist, and Prompt Engineer — all in one.

Give it a project idea. Get back a complete system design package.

---

## ⚡ Quick Start

```bash
# 1. Install the skill (user level — available across all projects)
git clone https://github.com/AbdallaElradi/senior-saad.git
cp -r senior-saad ~/.claude/skills/senior-saad

# 2. Open Claude Code in any project
claude

# 3. Activate SAAD
# Just describe your idea:
```

```
SAAD — I want to build a marketplace for freelancers
```
